# HPPsuPowerpoleOutput

This is a PCB designed to connect to the backplane of a HP server HSTNS-PD18 or HSTNS-PL18 power supply and provide powerpole outputs.
Please see https://www.qsl.net/gm0onx/SMPU.htm for instructions on how to modify these power supplies to increase the main output up to 13.5V.

# Parts
* 64 pin 2.54mm pitch edge connector. I got a pack of 5 from Aliexpress for about £7.50
* D1/2 standard 5mm led's
* Vertical pcb mount powerpole contacts and shells
* R1 - 22k Resistor
* R2/3 - 2k2 Resistor
* 3mm SPDT slide switch (e.g https://amzn.eu/d/0gzYgMrN)
